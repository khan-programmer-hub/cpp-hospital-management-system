# 🏥 C++ Hospital Management System

A simple Hospital Management System built using **C++ and Object-Oriented Programming (OOP)**.  
This project demonstrates basic CRUD operations and simulates real-world hospital record management.

---

## ✨ Features

- Add new patient records  
- Display all patient records  
- Search patient by ID  
- Delete patient record  
- Clean menu-driven interface  

---

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming (OOP)
- Arrays
- Console Application

---

## 📌 Project Purpose

This project was created to practice:
- OOP concepts in C++
- Problem-solving skills
- Basic data management system design
- Menu-driven programming logic

---

## 🚀 How to Run

1. Open the file in any C++ IDE (CodeBlocks / VS Code)
2. Compile the program
3. Run the executable
4. Use the menu options
